Librairie olympicWeather

Le package olympicsWeather permet à l’utilisateur d’obtenir les prévisions météorologiques des sites olympiques comme la température ressentie et attendue ou encore les précipitations.
L’utilisateur a le choix entre entrer une adresse textuelle et écrire directement les coordonnées gps (latitude et longitude).
A la fin, l’utilisateur obtient un fichier json avec toutes les informations.

Installation de la librairie

Pour installer la librairie, l’utilisateur doit avoir préalablement installé la librairie “remotes” puis installer le package olympicsWeather, relié à ma page github comme suit :

install.packages("remotes")
remotes::install_github("https://github.com/jas056/olympicsWeather/tree/master")
